• Samsung M30 :

1. Paramètres > À propos du téléphone > Informations sur le logiciel
2. Cliquer 7 fois sur "Numéro de version" ==> tu auras le message "Mode développeur activé"
3. Retourner aux Paramètres > Options pour les développeurs > Sélectionner application de localisation fictive > GPS Emulator

• Redmi 9 :

1. Paramètres > À propos du téléphone 
2. Cliquer 7 fois sur "Version MIUI" ==> tu auras le message "Vous êtes maintenant un développeur"
3. Retourner aux Paramètres > Paramètres supplémentaires > Options pour les développeurs > Sélectionner application de localisation fictive > LocaShift

• Realme UI Android :

1. Paramètres > À propos du téléphone 
2. Cliquer 7 fois sur "Version" (ou: "Numéro de build") ==> tu auras le message "Vous êtes maintenant un développeur"
3. Retourner aux Paramètres > Paramètres supplémentaires > Options pour les développeurs > Sélectionner application de localisation fictive > LocaShift
